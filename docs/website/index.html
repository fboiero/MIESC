17 Specialized Agents

  • StaticAgent - Slither, Solhint, Surya
  • AderynAgent - Ultra-fast Rust analyzer (Cyfrin)
  • WakeAgent - Python-based static analyzer
  • DynamicAgent - Echidna, Foundry fuzzing
  • MedusaAgent - Advanced fuzzing framework
  • SymbolicAgent - Mythril, Manticore
  • HalmosAgent - Symbolic testing
  • FormalAgent - Certora, Z3 verification
  • SMTCheckerAgent - Solidity built-in verifier
  • AIAgent - GPT-4 triage & correlation
  • OllamaAgent - Local LLaMA models
  • GPTScanAgent - AI vulnerability detection
  • InterpretationAgent - Finding analysis
  • RecommendationAgent - Remediation guidance (OpenZeppelin patterns)
  • PolicyAgent - ISO/NIST/OWASP compliance
  • CoordinatorAgent - Orchestration
  • CrewAICoordinator - Multi-agent collaboration
