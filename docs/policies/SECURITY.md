# Security Policy

The canonical, up-to-date security policy lives at:
**[../../.github/SECURITY.md](../../.github/SECURITY.md)**.

It covers how to report a vulnerability, the response timeline, the scope, and
the list of supported versions. This file previously held a separate copy with a
stale supported-versions table (and a "Last Updated: December 2024" footer) and
has been redirected to keep a single source of truth.

To report a vulnerability, email <fboiero@frvm.utn.edu.ar> with the subject
`[SECURITY] MIESC - Brief description`. Do not open a public issue.
