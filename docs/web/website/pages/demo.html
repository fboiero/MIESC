Step 1: Implement the SecurityAgent Interface

Simply copy it to the agents directory:

üé≠ Demo 3: Multi-Agent Orchestration

Run multiple agents in parallel for comprehensive security analysis:

Automatic Agent Selection

Let the orchestrator choose the best agents for your contract:

from src.orchestrator import AgentOrchestrator, SelectionCriteria # Initialize orchestrator orchestrator = AgentOrchestrator() orchestrator.initialize() # Define selection criteria criteria = SelectionCriteria( language='solidity', capabilities=[AgentCapability.STATIC_ANALYSIS], free_only=True, max_speed=AgentSpeed.MEDIUM ) # Select agents automatically agents = orchestrator.select_agents('MyContract.sol', criteria) print(f'Selected {len(agents)} agents') # Run analysis in parallel result = orchestrator.analyze('MyContract.sol', agents, parallel=True) # Get consolidated results print(f'Total findings: {result.get_summary()["total_findings"]}') print(f'Critical: {result.get_summary()["by_severity"]["CRITICAL"]}')

This runs all matching agents in parallel and consolidates results automatically!
