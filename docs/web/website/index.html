Quick Start (5 Minutes)

Docker (Recommended) 
# Pull and run
docker pull fboiero/miesc:latest
docker run --rm -v $(pwd)/contracts:/contracts miesc:latest \
    python xaudit.py --target /contracts/MyToken.sol

# Or use docker-compose
docker-compose up miesc
pip Installation 
# Install core tools
pip install slither-analyzer mythril manticore eth-wake

# Clone MIESC
git clone https://github.com/fboiero/MIESC.git
cd MIESC

# Verify installation
python scripts/verify_installation.py

# Run analysis
python xaudit.py --target examples/reentrancy_simple.sol
From Source 
# Clone repository
git clone https://github.com/fboiero/MIESC.git
cd MIESC

# Install Python dependencies
pip install -r requirements_core.txt

# Install Rust tools (optional, for Aderyn)
cargo install aderyn

# Install Node.js tools (optional, for Solhint)
npm install -g solhint

# Run verification
python scripts/verify_installation.py

Example: Analyze a Vulnerable Contract

vulnerable.sol
pragma solidity ^0.8.0;

contract VulnerableBank {
    mapping(address => uint) public balances;

    function withdraw() public {
        uint amount = balances[msg.sender];
        (bool success,) = msg.sender.call{value: amount}("");
        require(success);
        balances[msg.sender] = 0; // ‚ùå Too late!
    }
}
Analysis Result
CRITICAL Reentrancy Vulnerability

Location: withdraw() function, line 7

Type: SWC-107, OWASP SC-01, CWE-841

Description: External call before state update allows reentrancy attack

Detected by: Slither, Mythril, Manticore

üí° Suggested Fix: Update state before external call (Checks-Effects-Interactions pattern)
