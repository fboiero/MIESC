    MIESC Community¶

    Welcome to the MIESC community! This document describes how to connect with other users, contributors, and maintainers.

    Communication Channels¶

    GitHub (Primary)¶

    Channel Purpose Link
    Discussions Questions, ideas, show-and-tell GitHub Discussions
    Issues Bug reports, feature requests GitHub Issues
    Pull Requests Code contributions Pull Requests

    Discussion Categories¶

    Category Use For
    General Casual conversation, off-topic
    Q&A Questions about using MIESC
    Ideas Feature suggestions, brainstorming
    Show and Tell Share what you've built with MIESC
    Announcements Official updates from maintainers

    Getting Help¶

    Before Asking¶

    1. Search existing discussions and issues
    2. Check the documentation
    3. Run miesc doctor to diagnose common issues
    4. Review the GitHub Discussions Q&A

    Asking Good Questions¶

    • Be specific: Include error messages, versions, and steps to reproduce
    • Show your work: What have you already tried?
    • Provide context: What are you trying to accomplish?
    • Use code blocks: Format code and logs properly
