    Notas sobre el Formato¶

    Las referencias siguen el formato de la American Psychological Association (APA), 7ma edicion, conforme a las directrices del Publication Manual of the American Psychological Association (2020).

    Convenciones aplicadas:

    1. Articulos de conferencia: Se usa formato de capitulo en libro editado para proceedings
    2. Preprints (arXiv): Se incluyen como preprints con URL de arXiv
    3. Documentacion tecnica: Se cita como documento web con organizacion como autor
    4. Software/herramientas: Se cita como software con URL del repositorio
    5. Documentos gubernamentales: Se incluye organizacion y numero de publicacion
